Miwa.lol is built using a variety of open source software and projects. We wanted to give credit to the amazing work done by the open source community.

## Developers & Contributors

**Miwa.lol is mainly developed and maintained by [@CuteTenshii](https://github.com/CuteTenshii).** However, some features and improvements have been made by other contributors:
- [@oriionn](https://github.com/oriionn): Custom domains support, fixed typos, various improvements
- [@storcale](https://github.com/storcale): Various improvements and bug fixes

## What's Used

### Website

- [TypeScript](https://www.typescriptlang.org/)
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Drizzle ORM](https://orm.drizzle.team/) - Type-safe ORM for TypeScript
- [@tippyjs/react](https://tippyjs.com/) - Tooltips library
- [lucide-react](https://lucide.dev/) - Icons library
- [ua-parser-js](https://uaparser.dev/) - User-Agent parsing library
- [react-modal](https://reactcommunity.org/react-modal/) - Accessible modal dialog component
- [react-color](https://casesandberg.github.io/react-color/) - Color picker components
- [chart.js](https://www.chartjs.org/), [react-chartjs-2](https://react-chartjs-2.js.org/) - Charting libraries
- [react-toastify](https://fkhadra.github.io/react-toastify/) - Toast notifications library
- [@sentry/nextjs](https://docs.sentry.io/platforms/javascript/guides/nextjs/) - Sentry SDK for Next.js
- [stripe](https://github.com/stripe/stripe-node) - Stripe SDK for Node.js

### API

- [Go](https://golang.org/)
- [Gin](https://gin-gonic.com/)
- [gorm.io/gorm](https://gorm.io/) - ORM library for Go
- [github.com/discord/lilliput](https://github.com/discord/lilliput) - Image processing library, used for resizing and converting images to PNG/WebP format
- [github.com/fogleman/gg](https://github.com/fogleman/gg) - Used for generating Open Graph images
- [github.com/mileusna/useragent](https://github.com/mileusna/useragent) - User-Agent parsing library
- [github.com/redis/go-redis/v9](https://github.com/redis/go-redis) - Redis client for Go
- [github.com/golang-jwt/jwt/v5](https://github.com/golang-jwt/jwt) - JWT library for Go
- [github.com/u2takey/ffmpeg-go](https://github.com/u2takey/ffmpeg-go) - FFmpeg bindings for Go
- [github.com/pquerna/otp](https://github.com/pquerna/otp) - Generates and validates TOTP codes
- [github.com/google/uuid](https://github.com/google/uuid) - UUID generation
- [github.com/getsentry/sentry-go](https://github.com/getsentry/sentry-go) - Sentry SDK for Go
- [github.com/stripe/stripe-go/v82](https://github.com/stripe/stripe-go) - Stripe SDK for Go

- [Sentry](https://sentry.io/) - Error tracking
