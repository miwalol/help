# Spotify Embed

Embed a track, album, or playlist from Spotify on your page. This card uses the [Spotify Embeds](https://developer.spotify.com/documentation/embeds) to display the content.

For example, if you want to embed The Weeknd's "Starboy", you can create a card with the Spotify URL.
