The Bug Hunter badge is a special recognition for users who actively report bugs and issues in Miwa.lol.

## How to Earn the Bug Hunter Badge

To earn the Bug Hunter badge, you need to report bugs or issues that you find while using Miwa.lol.
Of course, it's not that simple, you need to report bugs that are not already known or reported by other users.

## Bug Reporting Guidelines

:::warning

We want to ensure that the Bug Hunter badge is awarded for significant contributions to improving Miwa.lol, **and not just to get the badge by reporting minor issues**.

:::

Those types of bugs are **NOT** considered valid for the Bug Hunter badge:
- **Already known bugs:** If the bug has already been reported by another user, or is already known to the Miwa.lol team, it does not count towards the Bug Hunter badge.
- **Minor issues:** Small issues that do not significantly affect the functionality or user experience of Miwa.lol are not considered valid bugs for the Bug Hunter badge.
- **Typos or grammar mistakes:** While these are "issues," they do not qualify as bugs for the Bug Hunter badge.

For a bug to be easily verifiable, it should:
- **Be reproducible:** The bug should be something that can be consistently reproduced by the Miwa.lol team.
- **Have a clear description:** The bug report should include a clear and concise description of the issue, steps to reproduce it, and any relevant screenshots or screen recordings.

We know we're asking a lot, but we want to ensure that the Bug Hunter badge is awarded for significant contributions to improving Miwa.lol (and not just for reporting minor issues just to get the badge).

## Reporting Bugs

The preferred way to report bugs is to use our [Discord server](https://discord.gg/miwa) in the dedicated forum channel. If you don't have Discord, you can also report bugs on our [GitHub](https://github.com/orgs/miwalol/discussions).

## Security Vulnerabilities

If you discover a security vulnerability in Miwa.lol, please report it to us immediately via:
- our [Discord server](https://discord.gg/miwa) **by opening a ticket**.
- contacting us via email at [security@miwa.lol](mailto:security@miwa.lol).


We take security seriously and will work with you to address the issue promptly. Please do not disclose the vulnerability publicly until it has been resolved.
