This page contains some common issues and their solutions.

### I don't receive the verification email

If you don't receive the verification email, please check your spam folder. Keep in mind that some email providers may block our emails, or delay them, such as Gmail.

If you still don't receive the email, please [contact our support team](/misc/contact) for assistance.

### It says "This user does not exist" but the username is already taken

This situation can happen in the following cases:
- The user is banned
- The user didn't finish the onboarding process
