Have a question? We tried to answer the most common ones here.

## What is Miwa.lol?

Miwa.lol is a biolink tool that allows you to create a customizable link in bio page.
It is designed to be simple and easy to use, with a focus on providing a great user experience. You can create a link in bio page that matches your brand and style, and share it with your audience.

## Is Miwa.lol free to use?

Yes, Miwa.lol is free to use. You can create a profile without any cost.
However, we do offer Premium features that require a subscription. These features are optional and not required to use the basic functionality of Miwa.lol.
