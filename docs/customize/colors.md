This pages explains what colors you can edit, and what is their utility.

## Accent color

This color is used for the `theme-color` meta tag and for the trail effect if you have the "Fairy Dust" cursor effect enabled.

:::info[What does the `theme-color` meta tag do?]

* Discord uses it to set the color of embeds.
* Safari uses it to adjust the status bar color.

:::

## Text color

This changes the color of the text on your profile.

## Background color

Sets the background color, which displays while your background image or video is loading, or if none is set.

## Badges color

:::note

Requires Premium.

:::

Changes the colors of your badges.

## Links color

If you enable "Monochrome Links," this option will change your links' color.

:::note

If "Monochrome Links" is disabled, adjusting this color will have no effect.

:::
