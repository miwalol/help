# Fonts

You can freely customize the font for your profile. Here’s a list of fonts you can choose from:

- Nunito
- Roboto
- Poppins
- Lacquer
- Fira Mono
- Lemon Milk
- Minecraft
- Noto Serif
- Noto Serif JP

:::tip

Most of these fonts are available on [Google Fonts](https://fonts.google.com/).

:::

## Custom Font

Premium users can upload their own custom font without needing to request us an addition.

To upload a custom font, simply click the upload button next to the search input.

![](./upload-font.png)

:::tip

Supported file formats: `ttf`, `otf`, `woff`, `woff2`, `eot`, `svg`.

:::
