Miwa.lol allows you to add various assets to customize your page. You can set a custom background, add a profile picture, and more to make your page unique.

## Uploading assets

Go to your [Dashboard > Customize](https://miwa.lol/dashboard/customize), then go to the *Assets* tab.

## Supported file types

| Type          | Supported file extensions                                        | Maximum size |
|---------------|------------------------------------------------------------------|--------------|
| Avatar        | `.jpg`, `.png`, `.gif`, `.webp`                                  | 10MB         |
| Audio         | `.mp3`, `.wav`, `.ogg`, `.flac`, `.aac`, `.m4a`                  | 30MB         |
| Audio cover   | `.jpg`, `.png`, `.gif`, `.webp`                                  | 10MB         |
| Background    | `.jpg`, `.png`, `.gif`, `.webp`, `.mp4`, `.webm`, `.mov`, `.avi` | 100MB        |
| Custom cursor | `.jpg`, `.png`, `.cur`, `.ico`                                   | 10MB         |
| Custom font   | `.ttf`, `.otf`, `.woff`, `.woff2`                                | 5MB          |
| Custom badge  | `.jpg`, `.png`, `.gif`, `.webp`, `.webp`                         | 10MB         |

<Aside type="info">
    The `.jpg` file extension is the same as `.jpeg`.
</Aside>
