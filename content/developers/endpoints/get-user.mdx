This endpoint allows you to retrieve information about a specific user by their username.

## Endpoint

```http
GET /api/user/{username}
```

### Path Parameters

| Parameter | Type   | Description                |
|-----------|--------|----------------------------|
| username  | string | The username of the user.  |
