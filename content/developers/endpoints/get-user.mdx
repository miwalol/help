Retrieve public profile information about a specific user by their username.

## Endpoint

```http
GET /api/users/{username}
```

### Path Parameters

<ParamTable>
  <Param name="username" type="string" required description="The username of the user." />
</ParamTable>

### Request Headers

| Header          | Value                    | Description                                  |
|-----------------|--------------------------|----------------------------------------------|
| Authorization   | `Bearer YOUR_API_KEY`    | Your API key. Required for all requests.     |

<Aside type="info">
  Don't have an API key yet? See [Creating an API Key](/developers/usage/creating-api-key/) to get started.
</Aside>

### Example Request

<Tabs>
  <Tab label="curl">
    ```bash
    curl https://miwa.lol/api/users/username \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Tab>
  <Tab label="JavaScript">
    ```js
    const res = await fetch('https://miwa.lol/api/users/username', {
      headers: { Authorization: 'Bearer YOUR_API_KEY' },
    });
    const user = await res.json();
    ```
  </Tab>
  <Tab label="Python">
    ```python
    import requests

    r = requests.get(
        'https://miwa.lol/api/users/username',
        headers={'Authorization': 'Bearer YOUR_API_KEY'},
    )
    user = r.json()
    ```
  </Tab>
</Tabs>

## Error Responses

| Status | Description                                                         |
|--------|---------------------------------------------------------------------|
| `401`  | Missing or invalid API key.                                         |
| `404`  | No user found with the given username.                              |
| `429`  | Rate limit exceeded. Wait before retrying.                          |
