# Contributing

**Want to help improve our help website?** We welcome contributions!

To contribute to the Miwa.lol help website, please follow the guidelines below. Your contributions will help us provide better documentation and support for our users.

## Requirements

- Familiarity with Markdown and basic HTML.
- Basic understanding of Docusaurus. You can find the [Docusaurus documentation here](https://docusaurus.io/docs)
- [Node.js](https://nodejs.org) or [Bun](https://bun.sh/) installed on your machine (we recommend using Bun because we use it for development).
- A code editor like [Visual Studio Code](https://code.visualstudio.com/) or any other of your choice.

## Guidelines

- **Write clear and concise documentation:**<br>
  We want to make it easy for users to understand how to use Miwa.lol and its features.
- **Use proper Markdown syntax:**<br>
  Headings, lists, links, and images should be formatted correctly.
- **Keep the content up-to-date:**<br>
  If you notice outdated information, feel free to update it!

## How to Contribute

1. Fork the repository, then clone it to your local machine.
2. Make your changes in the `docs` directory.
3. Test your changes locally by running `npm start` in the root directory.
   A new browser window should open with the local version of the help website
4. Commit your changes with a clear message describing what you have done.
5. Push your changes to your forked repository.
6. Create a pull request against the `master` branch of this repository.

We appreciate your contributions! If you have any questions or need help, feel free to [open an issue](https://github.com/miwalol/help/issues/new) in this repository.
